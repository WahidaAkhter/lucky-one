Question 1:How React works?
Answer: React is a JavaScript library which we can use to make user interface.React reads objects and uses them to create html element in virtual dom and then it gets synched in real dom.And then we can just show it to our ui.React is needed for single page applications.React does not commit state changes one after another instead it goes through virtual dom and make changes and compares that needed to be in the real dom and all does it in one single process.it is open source js library so it is easy to use to make readable user interface.
